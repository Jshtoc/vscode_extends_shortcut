## 2. 정리된 코드 만들기

- Beautify 검색 후 설치
- 설치 파일안 기능 기여도 안의 명령중 HookyQR.beautify 이름 복사
- vscode 위쪽 메뉴 상단부분에 파일 선택 후 기본설정안의 바로 가기 키 클릭
- HookyQR.beautify 붙여넣기 한 후 Beautify selection 부분의 키 바인딩 클릭
- 원하는 커맨드 입력 ex) 'ctrl + alt + l '
- 적용하는 방법 = 원하는 부분 드래그 후 커맨드키 입력
